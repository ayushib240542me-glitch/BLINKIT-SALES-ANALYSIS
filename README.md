# BLINKIT-SALES-ANALYSIS
# 🛒 Blinkit Sales Analysis — Python EDA Project

## 📌 Business Problem
Blinkit (India's last-minute delivery app) has sales data across 
multiple outlet types, sizes, and item categories. The goal is to 
identify what drives revenue, which outlets perform best, and where 
business opportunities exist.

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab
- Dataset: Kaggle — Blinkit Grocery Data

## 📊 Key Business Questions Answered
1. Which item types generate the highest total sales?
2. How does outlet size and location tier affect revenue?
3. What is the relationship between item fat content and sales?
4. Which outlet type has the highest average sales?
5. How does outlet establishment year correlate with performance?
